Orchestration tool for administration tasks.

```sh
$ apt-get install python-mysqldb python-pip
$ pip install -r requirements.txt
```


## Galera Cluster

Check the status of the galera cluster

```sh
$ ./server-compose check -h ip1 -h ip2 -h ip3
```
